How I Fell In Love With Code

It all started when I was in 4th grade of elementary school. I was looking for a way to speed up my old Samsung Galaxy Grand Neo phone.

After some Googling I found an article that talks about rooting and tweak. I looked into it, and it explains about using a simple Bash shell script to modify some system parameters, e.g. Low Memory Killer limit and CPU governor setting. I tried it, and... it almost did nothing. But the thing is, I instantly fell in love with the code!

I then learned Bash shell, and tried to make my own tweak. Later in middle school, I learned more programming languages like Python and C, although just some basics. Hey, it was a good start!
